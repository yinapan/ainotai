# AI 检测标注集

这个目录用于放置本地评测样本，帮助校准 AI 美术资源检测阈值。

## 目录

- `ai_generated/`：确认由 AI 生成或 AI 强辅助生成的图片。
- `human_made/`：确认由人工制作的图片。

## 使用建议

1. 保持文件名可追溯，例如 `source_tool_asset_type_001.png`。
2. 尽量覆盖 `_D`、`_N`、`_MADS`、`_E`、`_H` 等贴图类型。
3. 不要把真实项目敏感资源提交到 Git。
4. 每次新增样本后，可以用该目录跑离线扫描，再对比预测结果和目录标签。

## Git 规则

本目录默认忽略所有图片和样本文件，只提交目录结构和说明文档。
